## Summary

Protocol Labs needed to evolve a pre-agentic system for a new generation of agentic workflows. The work focused on improving throughput while keeping the security posture intact.

## Approach

- **Architecture modernization** - reviewed the existing system boundaries and adjusted the setup for higher-throughput agentic execution.
- **Deployment and operations** - improved the surrounding deployment and runtime setup across TypeScript, AWS, and Vercel.
- **Security preservation** - kept security controls central to the workflow changes, so velocity did not come at the cost of operational risk.

## Outcome

The system was prepared for agentic workflow adoption with a stronger operating setup and a clearer path to scale securely.
